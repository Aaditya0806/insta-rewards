How it works

Scratch, Scan & Win Instantly

Scratch & Reveal

Consumers scratch off a protective layer on the product packaging to reveal a secure, encrypted QR code. This layer ensures that the code remains tamper-proof until purchased by the end user.

Scan the Code

Using a mobile phone, consumers scan the QR code to trigger instant, secure validation. This step prevents unauthorized access and confirms that the code has not been duplicated or compromised.

Instant Rewards via UPI.

Upon successful validation, the system initiates an instant UPI payout directly to the consumer’s bank account. The confirmation screen provides immediate feedback, enhancing engagement through the satisfaction of instant rewards.
